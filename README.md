# Аура — сайт модельного агентства

Одностраничный сайт (лендинг) на **Vite + React**. Тёмная чёрно-белая («люкс») стилистика,
плавные анимации появления и карусель с фотографиями моделей.

## Запуск

Нужен установленный **[Node.js](https://nodejs.org) 18+** (проверить — `node -v`).

```bash
npm install      # установить зависимости (один раз)
npm run dev      # запустить локально — откроется на http://localhost:5173
```

Сборка для публикации:

```bash
npm run build    # результат появится в папке dist/
npm run preview  # посмотреть собранную версию локально
```

Папку `dist/` можно залить на любой статический хостинг: GitHub Pages, Netlify, Vercel, Cloudflare Pages и т.д.

## Что нужно заменить (заглушки)

Всё редактируется в одном файле — [`src/data/content.js`](src/data/content.js):

| Что               | Переменная       | Текущее значение (заглушка)     |
| ----------------- | ---------------- | ------------------------------- |
| Ссылка на Telegram | `TELEGRAM_URL`  | `https://t.me/your_username`    |
| Почта             | `EMAIL`          | `info@aura-models.ru`           |
| Телефон           | `PHONE`          | `+7 (916) 374-22-89`            |
| Города            | `CITIES`         | список из 9 городов             |
| Тексты вакансии   | `ABOUT`          | —                               |

> При смене телефона поправьте также `PHONE_HREF` (формат `tel:+7XXXXXXXXXX`, без скобок и пробелов).

## Фотографии моделей

Положите **6** фотографий в папку `public/images/models/` с именами:

```
model-1.jpg … model-6.jpg
```

- Ориентация — вертикальная (≈ 3:4), формат `.jpg`.
- Пока файлов нет, на их месте показываются заглушки «Фото 1 … 6».
- Изображения автоматически отображаются в чёрно-белом виде (под стиль сайта).
- Хотите другой формат (`.png`/`.webp`) — поправьте расширения в `src/data/content.js`.

## Структура

```
src/
  main.jsx              — точка входа
  App.jsx               — сборка секций (их порядок на странице)
  data/content.js       — ВСЕ тексты, ссылки, города, фото
  components/
    Hero.jsx            — крупная надпись «Аура»
    Gallery.jsx         — карусель с фото
    About.jsx           — текст вакансии
    Offices.jsx         — города + кнопка «Работать с нами»
    Contact.jsx         — почта + телефон с флагом РФ
    Footer.jsx          — ссылки подвала
    Reveal.jsx          — анимация появления при прокрутке
    RussianFlag.jsx     — флаг РФ (SVG)
  styles/index.css      — все стили
```
